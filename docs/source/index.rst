Welcome to the documentation for ortho_seqs!
============================================


Background:
===========

This documentation accompanies the ortho_seqs python command line tool that computes multivariate tensor-based orthogonal polynomials based
on DNA, RNA or protein sequence data and maps phenotypic information onto the sequence space.
